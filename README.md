# WorkoutCRM
WorkoutCRM - амбициозный проект с открытым исходным кодом.

**Проект находится на самой ранней стадии разработки**

### Что из себя представляет? 
WorkoutCRM - это платформа для спортсменов и тренеров.

+    Redux
+    SCSS
+ Backend:
+    Python
+    Flask/Fast API
+    MySQL
+    Swagger
+ Server:
+    Nginx
+ VCS:
+    GitHub
